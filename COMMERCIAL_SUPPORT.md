# Commercial Support / 商业支持

This repository is open-source. Paid support is available for narrowly scoped, reproducible Flutter-to-native integration problems.

本仓库保持开源，同时提供边界明确的 Flutter 与原生 SDK 集成付费支持。

## Supported scope / 可支持范围

- Flutter `MethodChannel`, `EventChannel`, and `PlatformView` debugging;
- Android Kotlin/Java, Gradle, AGP, and native SDK integration;
- iOS Objective-C/Swift, CocoaPods, Xcode, and native SDK integration;
- callback bridging, lifecycle issues, dependency conflicts, and build failures;
- Pangolin/Pangle content or advertising SDK integration when the requester has lawful SDK access and test credentials.

可处理 MethodChannel、EventChannel、PlatformView、Kotlin/Java、Objective-C/Swift、Gradle、CocoaPods、Xcode、回调桥接、依赖冲突和编译报错。涉及穿山甲或其他私有 SDK 时，需求方需提供合法授权的测试条件。

## Service options / 服务方式

### Root-cause diagnosis / 根因诊断 — CNY ¥99

For one reproducible issue: up to 45 minutes of log, configuration, and critical code-path review, followed by a written diagnosis and action checklist. This option does not include a code patch.

针对 1 个可稳定复现的问题：材料齐全后进行最多 45 分钟日志、配置和关键代码路径排查，交付书面根因判断与处理清单；不包含修改代码。

### Narrow bug fix / 单点修复 — CNY ¥599

For one narrow, reproducible issue: root-cause analysis, the smallest safe patch, source or patch delivery, change notes, verification steps, and one acceptance correction for the same issue in the agreed environment.

针对 1 个单一、可复现问题：原因定位、最小安全修复、源码或补丁、变更说明、验证步骤，以及约定环境下同一问题的 1 次验收修正。

If a ¥99 diagnosis is upgraded to the ¥599 fix on the same day, the diagnosis fee is fully credited and only ¥500 remains.

同日从 ¥99 诊断升级到 ¥599 修复时，诊断费全额抵扣，只补 ¥500。

Larger integrations, multi-module upgrades, new features, or dual-platform migrations are quoted separately after scope and acceptance criteria are agreed in writing.

完整 SDK 接入、多模块升级、新功能或双端迁移需先书面确认范围、交付物和验收标准后单独报价。

## Before contacting / 联系前请准备

- complete error logs or screenshots;
- `flutter doctor -v`;
- Flutter, Dart, Android Studio, Xcode, Gradle/AGP, and CocoaPods versions as applicable;
- shortest reproduction steps and expected behavior;
- a redacted minimal project or the relevant code path.

请准备完整日志、`flutter doctor -v`、相关工具链版本、最短复现步骤、预期结果，以及脱敏后的最小项目或相关代码。

## Safety and boundaries / 安全边界

- No passwords, signing private keys, payment credentials, or unrelated production data are requested.
- No work involving piracy, bypassing platform review or risk controls, unauthorized code, or unlawful SDK access.
- App Store or marketplace approval is never guaranteed.
- Device-specific issues require lawful access to the necessary hardware or a requester-operated test session.

不会索取密码、签名私钥、支付凭据或无关生产数据；不接盗版、绕过审核/风控、来源不明代码或未经授权的 SDK；不保证应用商店审核结果；真机问题需要合法测试设备或由需求方操作的测试会话。
